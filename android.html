Common Commands

// Install APK via adb
adb install -r app/build/outputs/apk/debug/app-debug.apk

// Build release AAB
./gradlew bundleRelease

// Generate signed APK (example)
./gradlew assembleRelease

// List connected devices
adb devices -l

Helpful Tips

  • Use Jetpack libraries for lifecycle, navigation and paging.
  • Prefer Android App Bundle (.aab) for Play Store uploads.
  • Automate signing and upload with GitHub Actions / Fastlane.
  • Enable ProGuard / R8 for release builds.
